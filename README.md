# 📊 SampleDatasets

Welcome to the **SampleDatasets** repository! This collection offers a variety of datasets in CSV and Excel formats, curated to support students and enthusiasts in practicing data analysis, visualization, and machine learning techniques.

## 📁 Repository Contents

The repository includes the following datasets:

- **1.02. Multiple linear regression.csv** — For practicing multiple linear regression.
- **CardioGoodFitness-1.csv** — Fitness equipment data; ideal for classification and regression.
- **Cars (1).csv** — Automotive data useful for regression analysis.
- **Churn_Modelling.csv** — Customer churn data for classification.
- **Copy of sonar data (1).csv** — Sonar signal data; binary classification.
- **Country-data.csv** — Socio-economic indicators; good for clustering/classification.
- **Employeers.csv** — HR analytics dataset.
- **Eye.csv** — Medical dataset related to eye measurements.
- **Global Superstore.xlsx** — Sales data for time series and market analysis.
- **HousePrices.csv** — Real estate pricing data for regression models.
- **Mall_Customers.csv** — Customer segmentation for clustering.
- **Metabolic Syndrome.csv** — Health dataset for classification.
- **Salaries (1).csv** — Salary-related data for regression and EDA.
- **Solar.csv** — Solar energy production; time series.
- **TaxiFare.csv** — Taxi fare prediction dataset.
- **World Happiness Report.csv** — Global happiness data; regression/correlation.
- **airline-passenger-traffic.csv** — Airline traffic; time series.
- **auto-mpg.csv** — Automobile fuel efficiency; regression.
- **bank-200520-125513.csv** — Banking data for classification.
- **brdata.csv** — Business data for general analysis.
- **concrete.csv** — Concrete strength prediction via regression.
- **data_clean.csv** — Cleaned sample dataset.
- **framingham.csv** — Framingham Heart Study data.
- **h1n1_vaccine_prediction.csv** — H1N1 vaccine prediction challenge data.

## 🎯 Usage

These datasets are intended for educational and research purposes. Use them to:

- Practice data preprocessing and cleaning.
- Build and evaluate machine learning models.
- Conduct exploratory data analysis (EDA).
- Understand domain-specific data structures.
